# Expense Tracker

## Overview
Expense Tracker is a Python-based application that helps users record and manage their daily expenses. It provides a simple way to monitor spending habits and calculate total expenditures.

## Features
- Add expenses
- Categorize expenses
- View expense records
- Calculate total expenditure
- User-friendly menu-driven interface

## Technologies Used
- Python

## Project Structure
ExpenseTracker/
│
├── expense_tracker.py
└── README.md

## How to Run

1. Clone the repository:
   git clone <repository-link>

2. Navigate to the project directory:
   cd ExpenseTracker

3. Run the program:
   python expense_tracker.py
